## üîÑ Keeping Documentation Updated

The interactive documentation (Swagger UI, ReDoc, OpenAPI schema) is automatically generated from the code. When the API changes:

1. Update `@extend_schema` decorators in `api/views.py`
2. Update serializers in `api/serializers.py`
3. Restart the server
4. Documentation updates automatically

To regenerate the OpenAPI schema file:
```bash
python manage.py spectacular --file api_schema.yml
```
